Lines 23-42: check_link(url) Function

def check_link(url):
    try:
        response = requests.get(url, timeout=10)
        return response.status_code, "Success" if response.status_code == 200 else "Failed"
    except requests.exceptions.RequestException as e:
        return None, str(e)

This function performs a network check to the url:

  • Tries an HTTP GET request with a 10-second timeout.
  • Returns the HTTP status code and a "Success" if status is 200 else "Failed".
  • Catches any exceptions (e.g., timeout, connection error) and returns the exception message.
It's used to quickly verify external link availability before proceeding.
